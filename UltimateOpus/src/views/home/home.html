Generators

  • yo dan:view create Backbone View class (with Template, Less file and generate require module)
  • yo dan:model create Backbone Model class
  • yo dan:collection create Backbone Collection class (with Model)

Commands

  • grunt to launch your project with watch and livereload
  • grunt build to build a deploy version of your project
  • grunt seo to generate seo files
  • grunt seo-preprod to generate seo files for preprod version

Enjoy coding! - Yeoman
