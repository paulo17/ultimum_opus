==========

Ultimum Opus

=======

It's a web project created by HETIC's students - P2017 during December

## This project is based on Framework DAN

Dependencies :
- RequireJS
- Yeoman
- BackboneJS
- jQuery
- Underscore
- Grunt

## API
- RoR API Ultimum Opus - JSON output (Simple Get for now, no Set)

------------

## To start this API

Please write command line :

```
    $ cd API_UltimumOpus
```

and after

```
    $ bundle install
```

## To start this project

Please write command line :

```
    $ cd UltimumOpus
```

and after

```
    $ npm install
```

Have fun ! :)
